## 设计作用

Tiny Item Animations 为 ProjectNautic 提供物品拾取与插入的缩放动画，直接服务于"UI 愉悦感"设计支柱。当玩家拾取物品或将物品插入容器时，物品会先放大再缩回正常尺寸，为原本瞬间完成的操作增加轻微的视觉弹性。

该模组与 Inventory Particles、Extra Sounds、Smooth Swapping 共同构成 ProjectNautic 的物品栏体验层：Tiny Item Animations 负责缩放动画，Inventory Particles 负责粒子轨迹，Extra Sounds 负责音效，Smooth Swapping 负责位移动画。

## 玩家可见行为

- **可发现性**：玩家拾取掉落物或将物品放入容器/物品栏时，物品图标会播放一个短暂的放大-缩小动画。
- **交互**：无需主动操作；动画在物品状态变化时自动触发。
- **进度/奖励**：无内置进度；动画作为物品操作的视觉奖励。
- **摩擦/失败**：动画速度过快可能几乎不可见，过慢则可能让玩家感觉操作延迟；过高的拾取缩放可能遮挡邻近格子。

## 配置意图

`config/tia-client.toml` 当前保留了模组默认值，意图是让动画自然但不过度夸张：

- **`animation_speed=0.5`**：默认动画速度，使拾取/插入动画既可见又不拖沓。
- **`pickup_scale=1.399999976158142`**：默认拾取时最大放大到约 1.4 倍，提供明显的视觉反馈但不至于完全遮住格子。

整合包未对这两个参数做自定义，说明当前设计意图是接受模组作者预设的动画节奏，作为对原版物品栏操作的轻量润色。
