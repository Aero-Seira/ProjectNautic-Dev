## 设计作用

MidnightLib 是 ProjectNautic 整合包的轻量级配置库模组，为依赖它的模组（如 CullLeaves）提供统一的配置屏幕、配置命令与配置序列化能力。它属于 `library` 分类：对普通玩家不可见，是其他模组正常运行的运行时依赖。

在 ProjectNautic 的技术栈中，MidnightLib 支撑**UI 一致性**与**性能优先**的设计支柱：通过统一的配置界面减少各模组自定义配置屏幕带来的视觉碎片化，同时降低模组开发者重复实现配置系统的成本。

## 玩家可见行为

- **可发现性**：普通玩家不会直接看到 MidnightLib 的名称，但会在依赖它的模组（如 CullLeaves）的配置界面中体验到统一的配置屏幕风格。
- **交互**：无直接交互。配置能力通过依赖模组暴露。
- **进度/奖励**：无。
- **摩擦**：无额外摩擦。
- **失败状态**：若 MidnightLib 缺失或被错误版本替换，依赖它的模组将无法加载或配置界面失效。

## 配置意图

`config/midnightlib.json`：

```json
{
  "config_screen_list": "MODMENU"
}
```

- **`config_screen_list: MODMENU`**：指定配置屏幕列表的显示模式为 MODMENU 风格。这是 MidnightLib 自身的默认行为，未做额外覆盖。

该配置文件仅控制 MidnightLib 自身的行为，不影响依赖它的模组配置。各模组的具体配置仍保留在各自的配置文件中（如 `config/cullleaves.json`）。

MidnightLib 还提供了配置命令与配置同步机制，使得依赖它的模组可以通过统一的方式暴露设置项。当前未对这些高级功能做自定义配置，保持最小化干预。
