# I18nUpdateMod 自动汉化更新

## 记录

- 类型：mod
- 状态：active
- 证据可信度：verified
- 来源路径：`mods/[自动汉化更新] I18nUpdateMod-3.7.0-all.jar`
- 标识与版本：`i18nupdatemod` `3.7.0`
- 加载器或包格式：NeoForge 21.1.235（Minecraft 1.21.1）

## 设计作用

I18nUpdateMod 自动下载并应用社区维护的 Minecraft 模组汉化资源包，确保中文玩家在包内看到一致的中文翻译。它支撑整合包**本地化体验**的设计意图，减少玩家手动寻找、下载、放置汉化资源包的操作，同时保证新增模组上线后翻译能及时更新。

该模组以 Fabric mod 形式打包，并通过 NeoForge ModLauncher 的 `ITransformationService` 加载；实际运行环境为 CLIENT。

## 玩家可见行为

- **可发现性**：启动游戏时，模组会自动检查并下载最新汉化资源包；进入游戏后，模组物品、界面、进度等显示为中文。
- **交互**：玩家无需手动操作；汉化包会自动加入已选资源包列表并启用。
- **进度/奖励**：无直接关联。
- **摩擦/失败**：若网络连接失败或汉化包仓库不可用，可能无法下载最新翻译；部分新模组可能尚未被汉化包覆盖。

## 集成关系

- **依赖**：无显式依赖声明；依赖 NeoForge / Fabric 加载机制与网络环境。
- **冲突**：与手动覆盖同名翻译键的资源包可能产生覆盖顺序冲突。
- **协作**：与 `resourcepacks/Minecraft-Mod-Language-Modpack-Converted-1.21.1.zip` 协作，后者为本地已转换的汉化资源包；I18nUpdateMod 负责保持其最新。
- **配方/标签**：无。
- **任务**：无。
- **世界生成**：无。
- **UI**：无游戏内 UI 变更。
- **资源**：需要网络下载汉化资源包；本地 `resourcepacks/` 目录会被更新。
- **脚本**：无 KubeJS 交互。
- **加载顺序假设**：作为转换服务尽早加载；汉化资源包在资源包列表中的优先级由模组自行管理。

## 配置意图

该模组无本地配置文件。其设计意图为：在客户端启动时静默检查并下载最新版社区汉化资源包，使整合包的中文翻译保持最新，无需玩家干预。

## 兼容性与性能

- **客户端/服务端范围**：CLIENT ONLY。`fabric.mod.json` 中 `environment` 为 `client`。
- **存档影响**：无。
- **已知不兼容**：无。
- **资源成本**：仅在启动时发起一次网络请求并下载/更新资源包；对帧率与内存无持续影响。
- **缓解措施**：若汉化包下载失败，游戏仍可使用内置或本地资源包 fallback；网络环境受限时玩家可手动管理汉化包。

## 风险与开放问题

1. 汉化包仓库可用性与网络稳定性直接影响玩家首次启动体验；需监控官方仓库状态。
2. 部分模组自定义字体或渲染方式可能与社区汉化包中的字符宽度不一致，导致 UI 文字截断。
