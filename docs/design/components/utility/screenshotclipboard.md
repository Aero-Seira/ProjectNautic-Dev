- 加载器或包格式：NeoForge 21.1.235（Minecraft 1.21.1）

## 设计作用

Screenshot to Clipboard 在玩家按下 F2 截图时，将图片数据自动复制到系统剪贴板。它支撑整合包**社交与分享便利**的设计意图，让玩家无需手动打开截图文件夹即可直接粘贴到聊天、Discord 或文档中，降低分享游戏画面的操作成本。

## 玩家可见行为

- **可发现性**：玩家截图后，系统剪贴板中已包含该截图；粘贴到外部应用时即可看到图片。
- **交互**：无额外游戏内交互；截图流程与原版本完全一致。
- **进度/奖励**：无直接关联。
- **摩擦/失败**：在某些平台或 Java 版本下，剪贴板写入可能失败；部分杀毒软件可能阻止剪贴板访问。

## 配置意图

该模组无独立配置文件。其行为为：只要安装即生效，所有 F2 截图自动写入系统剪贴板，同时保留原文件到 `screenshots/` 目录。

## 兼容性与性能

- **客户端/服务端范围**：标记为 BOTH，但实际功能仅在客户端截图时触发；服务端安装主要用于版本同步。
- **存档影响**：无。
- **已知不兼容**：无。
- **资源成本**：仅在截图时将图片写入剪贴板，几乎无持续性能开销。
- **缓解措施**：若玩家不需要此功能，可移除该模组；它不改变截图文件保存逻辑。
